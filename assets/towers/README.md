## Sprite Requirements

- **Format**: PNG with transparency
- **Resolution**: 64x64 pixels (base size)
- **Color Depth**: 32-bit RGBA
- **Style**: Flat design or simple 3D with clear silhouettes
- **Upgrades**: Each tower has 3 levels - consider visual progression (size, detail, effects)

## Asset Optimization

Follow the IMAGE_OPTIMIZATION_GUIDE.md in the assets root directory:
- Use PNG format for sprites with transparency
- Keep file sizes under 50KB per sprite
- Use simple, clear designs that work at small sizes
- Ensure good contrast for visibility on various backgrounds
